# Rice Leaf Disease Detection

## Overview

Rice crops are vulnerable to various diseases that can significantly impact agricultural productivity. This project focuses on automatically detecting and classifying major rice leaf diseases using machine learning and computer vision techniques.

## Problem Statement

Develop a machine learning model capable of classifying rice leaf diseases into the following categories:

* Leaf Smut
* Brown Spot
* Bacterial Leaf Blight

The project also includes exploratory data analysis, data augmentation techniques, model comparison, and performance evaluation.

## Dataset

The dataset consists of 120 disease-infected rice leaf images.

* 40 images of Leaf Smut
* 40 images of Brown Spot
* 40 images of Bacterial Leaf Blight

## Project Workflow

1. Data Collection
2. Exploratory Data Analysis (EDA)
3. Image Preprocessing
4. Data Augmentation
5. Model Training
6. Model Evaluation
7. Model Comparison
8. Performance Analysis

The trained model successfully classified rice leaf diseases and achieved strong classification performance across all classes.
